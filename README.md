1. **매주 금요일 오전** — 스터디장이 주제 2개 수집 (어려웠던 것 1 + 중요한 것 1)

3. **사다리타기** — 발표자 배정 (각 주제당 1명)
4. **모임 전날까지** — 발표 자료 준비 (15분 이내)
5. **모임 날** — 발표 + Q&A + GitHub 업로드 ✅
